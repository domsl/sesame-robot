# CAD Files

This directory contains the 3D CAD models and STL files for 3D printing the Sesame Robot body.

## File Organization

Store CAD files with descriptive names indicating the part:
- Body parts
- Leg components
- Mounting brackets
- Hardware fixtures

## File Formats

- **Source files**: Native CAD format (STEP, IGES, or CAD software files)
- **STL files**: Ready-to-print 3D models
- **Assembly files**: Full robot assembly for reference

## Printing Guidelines

- **Material:** PLA / PLA+
- **Infill:** 15-20%
- **Supports:** Minimal (check individual file notes)
- Refer to the Build Guide in `/docs/build-guide/` for specific orientation guidelines
