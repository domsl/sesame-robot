# Bill of Materials (BOM)

This directory contains the complete bill of materials for building the Sesame Robot.

## BOM Files

Store BOM files in various formats:
- CSV spreadsheets
- Excel files
- Markdown tables
- Interactive HTML BOMs

## BOM Categories

The BOM should include:

### Electronics
- Microcontroller
- Servo motors (quantity: 8)
- Servo driver board
- OLED display
- Power supply
- Cables and connectors

### Hardware
- Screws and fasteners
- Mounting hardware
- Standoffs

### Materials
- 3D printing filament requirements
- Adhesives (if needed)

### Optional
- Sensors
- Additional components for modifications

## Supplier Information

Include supplier part numbers and links where possible to make sourcing easier.
