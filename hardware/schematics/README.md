# Schematics

This directory contains electronic schematics and wiring diagrams for the Sesame Robot.

## Contents

- Circuit diagrams showing connections between components
- Wiring diagrams for servo connections
- I2C bus wiring for PCA9685 and OLED display
- Power distribution schematics

## File Formats

- PDF for easy viewing
- Source files from schematic capture tools (KiCad, Eagle, etc.)
- PNG/SVG for quick reference

## Key Connections

- Arduino I2C pins to PCA9685 (SDA/SCL)
- PCA9685 channels 0-7 to servos
- OLED display via I2C
- Power supply distribution
