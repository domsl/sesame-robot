# Software

This directory contains software applications for controlling and interacting with the Sesame Robot.

## Directory Structure

- **web-ui/** - Web-based user interface for robot control
- **control-app/** - Desktop or mobile control applications

## Web UI

The web UI provides a graphical interface for controlling the robot via WiFi (if ESP32 is used).

Features:
- Button controls for all available poses
- Real-time status display
- Animation triggers

## Control Applications

Additional control software, utilities, and interfaces for:
- Desktop control apps
- Mobile applications
- API interfaces
- Testing tools

## Development

Contributions welcome for:
- Improved Web UI/UX
- New control interfaces
