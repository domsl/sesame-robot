# Documentation

This directory contains all documentation for the Sesame Robot project.

## Directory Structure

- **assembly/** - Step-by-step assembly instructions
- **build-guide/** - Complete build guide with photos and tips
- **images/** - Photos, diagrams, and illustrations
- **datasheets/** - Component datasheets and technical specifications

## Documentation Guidelines

When contributing documentation:
- Use clear, concise language
- Include plenty of photos and diagrams
- Provide step-by-step instructions
- List common issues and troubleshooting tips
- Keep documentation up-to-date with hardware/firmware changes

## Key Documents

- Assembly instructions for physical construction
- Electronics assembly and wiring
- Firmware installation guide
- Calibration procedures
- Troubleshooting guide
- FAQ
