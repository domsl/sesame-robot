# Build Guide

This directory contains the complete build guide for the Sesame Robot.

## Contents

The build guide should include:
- Complete parts list with sourcing information
- Detailed assembly instructions
- Wiring diagrams
- Firmware setup procedures
- Calibration guide
- Troubleshooting tips

## Printing Guidelines

### Material Settings
- **Material:** PLA / PLA+
- **Infill:** 15-20%
- **Supports:** Minimal (check individual file notes)

### Part Orientation
Specific orientation guidelines for each part to ensure:
- Optimal strength
- Minimal support material
- Best surface finish

Refer to individual part documentation in `/hardware/cad/` for specific recommendations.

## Build Time

Approximate build times:
- 3D Printing: varies by printer (typically 20-40 hours total)
- Assembly: 2-4 hours
- Electronics/Wiring: 1-2 hours
- Testing/Calibration: 1 hour

## Skill Level

This project is designed for makers of all skill levels:
- **Beginner:** Follow the guide step-by-step
- **Intermediate:** Customize and modify as desired
- **Advanced:** Use as a platform for advanced features
