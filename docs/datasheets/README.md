# Datasheets

This directory contains datasheets and technical documentation for components used in the Sesame Robot.

## Contents

Datasheets for:
- MG90 Servo Motors
- PCA9685 PWM/Servo Driver
- SSD1306 OLED Display
- Arduino boards (Uno, Nano, Mega)
- Power supplies and regulators
- Any additional sensors or components

## Organization

Organize by component type:
- **servos/** - Servo motor datasheets
- **controllers/** - Microcontroller datasheets
- **drivers/** - Motor/servo driver datasheets
- **displays/** - Display module datasheets
- **sensors/** - Sensor datasheets (if added)
- **power/** - Power supply and regulator datasheets

## File Naming

Use clear, searchable names:
- `MG90_servo_datasheet.pdf`
- `PCA9685_datasheet.pdf`
- `SSD1306_OLED_datasheet.pdf`

## Usage

Datasheets are helpful for:
- Understanding component specifications
- Pin configurations
- Electrical characteristics
- Operating limits
- Communication protocols
- Timing diagrams

## Sources

When adding datasheets, note the source and version to ensure accuracy.
