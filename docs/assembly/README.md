# Assembly Instructions

This directory contains detailed assembly instructions for building the Sesame Robot.

## Assembly Guides

Store assembly documentation in logical sections:
- Part preparation and cleanup
- Body assembly
- Leg assembly
- Electronics installation
- Final assembly and testing

## File Formats

- Markdown for text instructions
- PDF for printable guides
- Images in the `/docs/images/` directory

## Assembly Sections

### 1. Parts Preparation
- 3D printed part cleanup
- Support removal
- Test fitting

### 2. Mechanical Assembly
- Body assembly
- Leg construction
- Servo installation
- Hardware fastening

### 3. Electronics Assembly
- Wiring harness preparation
- Component mounting
- Connections and routing
- Power system setup

### 4. Final Assembly
- Integration testing
- Calibration
- First power-on

## Tips

- Take your time with assembly
- Test fit before gluing/fastening
- Keep track of hardware
- Refer to build videos for visual guidance
