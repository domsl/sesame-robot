# Images and Media

This directory contains images, photos, diagrams, and other media for documentation.

## Organization

Organize images by category:
- **assembly/** - Assembly step photos
- **wiring/** - Wiring diagrams and connection photos
- **build/** - Build process photos
- **completed/** - Photos of completed builds
- **diagrams/** - Technical diagrams and schematics
- **screenshots/** - Software/UI screenshots

## File Naming

Use descriptive names:
- `step_01_body_assembly.jpg`
- `wiring_diagram_i2c.png`
- `completed_front_view.jpg`

## Image Guidelines

- Use high-resolution images (at least 1920x1080 for photos)
- Ensure good lighting and focus
- Include scale references where helpful
- Use annotations for complex images
- Optimize file sizes for web use

## Formats

- **Photos:** JPG or PNG
- **Diagrams:** SVG (vector) or high-res PNG
- **Screenshots:** PNG
- **Videos:** Link to external hosting (YouTube, etc.)

## Attribution

If using images from community contributions, include attribution in filename or separate attribution file.
